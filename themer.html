Theme CSS generator

This tool is meant to help you create theme css for use in the Uniform jQuery plugin.

In order for this to work properly you must have already created a sprite image using the included template.

Additionally, the sprites must be in the same order as the template: selects, then checkboxes, then radios, then file inputs. This generator assumes you are following that pattern.

Generated code

Copy and paste this code into a text editor, and save it as uniform.YOURTHEMENAME.css

You will most likely have to adjust some additional properties in the presentation area before your theme looks perfect
